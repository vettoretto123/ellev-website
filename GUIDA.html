Come gestire il tuo sito

Guida semplice per aggiornare i contenuti e pubblicare online

// Il flusso di lavoro

Come fare le modifiche, passo per passo
  1. Fai doppio click su avvia-modifiche.bat — si apre automaticamente la finestra nera con il sito in anteprima, Visual Studio Code e il browser con il tuo sito già caricato.
  2. Modifica i contenuti — in Visual Studio Code apri src/_data/it.json per l'italiano o src/_data/en.json per l'inglese. Fai le modifiche e salva con Ctrl+S.
  3. Controlla il risultato — torna sul browser e aggiorna la pagina (tasto F5 oppure Ctrl+R). Le modifiche appariranno subito.
  4. Quando sei soddisfatta — chiudi la finestra nera. Poi fai doppio click su update.bat e premi INVIO.
  5. Aspetta 1-2 minuti e controlla il tuo sito online. Le modifiche sono pubblicate!
✏️
// Strumento 1
avvia-modifiche.bat

Questo è il file da cui si parte sempre. Con un solo doppio click avvia il sito in anteprima sul tuo PC e apre Visual Studio Code per modificare i testi — tutto in una volta.

Da Visual Studio Code puoi aprire i file src/_data/it.json e src/_data/en.json dalla colonna di sinistra, modificare i testi e salvare con Ctrl+S.

Quando usarlo: ogni volta che vuoi lavorare sul sito. È l'unico file che devi aprire per iniziare — fa tutto lui.
Lascia la finestra nera aperta mentre lavori. Se la chiudi, l'anteprima del sito si spegne.
🚀
// Strumento 3
update.bat

Questo file pubblica le tue modifiche online, rendendole visibili a tutti i visitatori del sito.

Come funziona: fai doppio click → si apre una finestra nera → premi INVIO → aspetta qualche secondo → fatto.

Quando usarlo: solo quando sei soddisfatta delle modifiche e vuoi che tutti le vedano. Dopo aver premuto INVIO, il sito online si aggiorna in 1-2 minuti.
Premi Q + INVIO per uscire senza pubblicare nulla, se hai aperto il file per sbaglio.

// I contenuti del sito

Cosa puoi modificare nei file di testo

Tutti i testi del sito si trovano in due file: src/_data/it.json (versione italiana) e src/_data/en.json (versione inglese). Li apri da Visual Studio Code, cambi il testo dopo i : e salvi.

Come aprire e modificare un file in Visual Studio Code

  1. Fai doppio click su avvia-modifiche.bat — si apre Visual Studio Code
  2. Nella colonna di sinistra, clicca sulla cartella src → _data
  3. Clicca sul file it.json (o en.json) per aprirlo
  4. Trova il testo che vuoi cambiare e modificalo
  5. Salva con Ctrl+S
  6. Torna sul browser e premi F5 per vedere le modifiche
Guida ai campi
meta_title
meta_description
Il titolo e la descrizione che appaiono nei motori di ricerca (Google) e quando condividi il link sui social. Es. "VOICE — Unlock Your Art"
nav_cta
cta_url
Il testo e il link del pulsante principale in cima alla pagina ("Prenota la Call Gratuita"). Cambia cta_url con il tuo link Calendly personale.
hero_title_*
hero_sub
Il titolo grande della prima schermata e il testo descrittivo sotto. "Sblocca la tua Voce. Libera la tua Arte." + il paragrafo introduttivo.
blocco_*_title
blocco_*_sub
I quattro blocchi nella sezione "Il Problema": Ansia da performance, Paura del giudizio, Autosabotaggio, Frustrazione.
pillar_*_title
pillar_*_body
Le cinque dimensioni del metodo VOICE: Identità, Relazioni, Paura del giudizio, Valore, Missione artistica.
Da fare
coach_bio_1
coach_bio_2
coach_quote
La tua presentazione personale: da dove sei partita, il tuo percorso come artista, le tue credenziali e una frase che ti rappresenta. Questi campi contengono ancora "[Da configurare]" — vanno compilati con i tuoi testi.
Da fare
nonperchi_items
L'elenco di chi non è adatto al percorso. Quattro punti da personalizzare con i tuoi criteri.
testi
Le testimonianze: nome, ruolo e testo di ogni persona. Puoi modificare quelle esistenti o aggiungerne di nuove. Ogni testimonianza è racchiusa tra { } con nome, ruolo e testo.
Da fare
faq
Le domande frequenti: ogni domanda ha una "q" (domanda) e una "a" (risposta). Scrivi le domande che ricevi più spesso.
social
I link ai tuoi profili social. Per attivare o disattivare un social cambia true (visibile) o false (nascosto). Instagram, Facebook e TikTok sono già configurati.
cta_url
Il link al tuo Calendly per prenotare la call gratuita. Aggiornalo se cambi il link di prenotazione. Appare in più punti della pagina — cambiarlo qui lo aggiorna ovunque.

Ricorda: le modifiche ai file di testo
